Why

和 AI 协作时，你有没有遇到这些问题

截图污染 Git 仓库

报错截图随手放进项目目录，结果被 AI 一起提交到 Git，仓库里满是 screenshot-xxx.png

长文本反复修改

给 AI 的提示词太长被缩略显示，只能复制出来改完再粘贴回去，来回折腾

临时文件散落各处

代码片段、日志、链接、文件路径散落在桌面和剪贴板，找不到也理不清

图片交付给 AI 的困境

想让 AI 分析截图，但截图放在工作目录会被误提交，左右为难

Features

核心功能

悬浮在桌面最上层的临时中转站，随时可用，用完即走。

文本收纳

粘贴后自动生成摘要标题，支持重命名，收起也能一眼识别内容。

  • 展开直接编辑，改完一键复制
  • 自动检测代码内容

图片收纳

截图 Ctrl+V 直接导入，存储在独立目录，不会进入 Git 仓库。

  • 复制图片内容 — 直接粘贴到聊天框
  • 复制图片路径 — 给 AI 读取

文件收纳

Ctrl+C 复制文件，Ctrl+V 导入；也支持直接拖拽。

  • 支持复制内容或路径
  • 支持 doc、xlsx、pdf 等格式

收纳 / 收藏 / 全部

  • 收纳 — 临时暂存，启动时自动清空
  • 收藏 — 长期保留，关机不丢
  • 全部 — 分类筛选文本/图片/文件

桌面常驻

  • 置顶模式 — 一键切换始终置顶
  • 最小化 — 收为桌面角落的猫咪
  • 全局快捷键 Alt+Shift+V
  • 系统托盘后台运行

个性化设置

  • 三套主题 + 跟随系统
  • 中英文字体分别设置
  • 中英文界面即时切换
  • HTTP/SOCKS5 代理更新
  • 开机自启
